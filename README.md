# Selenium Automation Setup

This project uses Selenium WebDriver with Chrome to automate browser interactions. It includes support for dynamic waits, stealth browsing, and automatic ChromeDriver management.

---

## 🚀 Features

- Automated browser control using Selenium
- ChromeDriver auto-install via `webdriver-manager`
- Stealth mode to reduce bot detection
- Explicit waits for stable element handling
- Exception handling for robustness

---

## 📦 Dependencies

Make sure you have Python 3.8+ installed.
